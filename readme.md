## Installing python3 in GNU/Linux or WSL 
---
Debian based Distributions
```bash
sudo apt install python3 
```
FreeBSD bases Distributions
```bash
sudo pkg install python3
```
Arch based Distributions
```bash
sudo pacman -Sy python3
```
